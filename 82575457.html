Цели

  • Заведение рекламных материалов маркетингом в PIMCore

As Is

  • Маркетинг заводит часть рекламных материалов (подарочные карты, реклама, упаковка), предоставляя информацию по шаблону товароведам
  • Частично вносит правки в карточку товара (функция перейдет товароведам)

To Be

  • Маркетинг заводит рекламные материалы в PIMCore
  • Для товара, который предназначен для автоматического добавления в заказ в качестве дополнительной выгоды при выполнении условия покупки (корзина ИМ Brocard.ua), необходимо указать Код назначения товара = PROMOTION и код типа товара = GIFTS
  • Для карточки товара, которая относится к акционным картам (пластик со ШК) не создавать код варианта
