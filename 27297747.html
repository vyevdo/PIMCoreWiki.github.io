Brocard : Импорт CSV файла (depricate)

Created by Illia Fedorovych, last modified by Dmytro Svietnoi on Nov 22, 2022

Нажмите ПКМ на "Home" в разделе "Products".

Затем выберите пункт "Product" в разделе "CSV Import".

В появившемся окне выберите файл для загрузки.

Откроется окно с настройками для импорта. На вкладке "CSV File Preview" можно увидеть предварительный просмотр файла.

Для того, чтобы перейти к настройкам колонок, нажмите "Column Configuration".

На этой вкладке отображаются настройки колонок и их соответствие к полям товара в Pimcore.

Для того, чтобы создать соответствие, перетяните поле товара из раздела "Class Attributes" (1) к соответствующей колонке файла в разделе "Selected Grid Columns" (2).

Также в случае необходимости, вы можете использовать операторы в разделе "Operators" (3) для дополнительного преобразования данных. Для получения детальной информации смотрите https://pimcore.com/docs/latest/User_Documentation/DataObjects/CSV_Import/Operators/index.html

Обратите внимание, что такие поля как "Тип товара", "Торговая марка" и подобные являются ссылками на объекты, которые существуют в системе Pimcore. Их значения можно указать в CSV файле как полный путь к объекту (например, тип товара "/Значение/Тип товара/activator"), или же задать уже после импорта при редактировании карточки товара.

Также, Вы можете сохранять и загружать настройки для импорта. Для того, чтобы загрузить сохраненные настройки, нажмите на кнопку "Load".

В появившемся окне выберите конфигурацию, которую хотите загрузить и нажмите "ОК".

Для сохранения настроек нажмите кнопку "Save".

Обратите внимание, что сохраняются данные не только из вкладки "Column Configuration", а также из вкладок "Resolver Settings", "CSV Settings" и "Save and Share".


В появившемся окне введите название для конфигурации и описание (необязательно), после чего нажмите "Save".

На этой вкладке отображаются настройки распознавания объектов. Здесь можно задать колонки, которые будут использованы для определения того, существует ли уже данный объект в системе.

Для детальной информации смотрите https://pimcore.com/docs/latest/User_Documentation/DataObjects/CSV_Import/Resolver/index.html

Для того, чтобы перейти к настройкам файла CSV, нажмите "CSV Settings".

На этой вкладке отображаются настройки файла CSV. Для того, чтобы перейти к параметрам конфигурации, нажмите "Save & Share".

На этой вкладке находятся настройки для сохранения и обмена конфигурации.

Для того, чтобы начать процесс импорта, используя текущие настройки, нажмите "Import".

После завершения процесса импорта, Вы получите сообщение об успешном либо неудачном завершении. Нажмите кнопку "ОК", чтобы перейти к результатам.

На вкладке "Import Report" находятся результаты последнего процесса импорта. В случае, если при импорте возникли проблемы, в колонке "Message" будет находиться описание ошибки.

Для того, чтобы открыть успешно импортированный товар, нажмите на значке "Open".

Вы можете удостовериться в успешном проведении импорта, проверив карточку товара на наличие необходимой информации.


Также вы можете проверить наличие импортированных товаров в полном списке, нажав на папку "Home" и проверив их наличие в списке.
