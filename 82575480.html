Цели

  • Автоматизировать список проверок при заведении новой продукции
  • Поддержка корректного ассортимента 

As it

  1. В Excel file есть связь "*Назначение (основное)"→ "*Тип товара"
  2. Торговая марка = текстовое поле
  3. Автоматично присвоюються такі дані:
  4. Категорія/категорії всіх рівнів (за сукупністю даних в пунктах 1-5). Крім товарів, для яких потрібно додатково вибрати категорію всіх рівнів, такі типи товарів виділені жовтим
  5. Область применения Тетра через тип товару. Крім товарів, для яких це визначає постачальник (такі товари виділені жовтим)
  6. Область применения NAV через тип товару
  7. Лінія на укр.мові для існуючих ліній (англ версію вибирає зі списку постачальник)
  8. Країна похождення (для існуючих брендів)
  9. Виробник (для існуючих брендів)
  10. Ознака імпортного товару (для існуючих брендів)
  11. Назва товару для PimCore

Предположения

To be 

  1. Настроить связь справочников "*Назначение (основное)"→ "*Тип товара" в PIM Core
  2. Сделать поле торговая марка выпадающим списком из существующих брендов. В слючае нового бренда добавить в выпадающий список значению New brand. При его выборе заполнить новый столбец рядом
  3. Добавить значения в поле "*Пол" = for boys; for girls, ...
  4. Стандартизировать поле объем(сделать выпадающий список с поиском)
