Руководство пользователя Pimcore для поставщика

Доступ к инструкциям в системе Pimcore



Авторизация

После того, как Вы будете зарегистрированы, Вы получите письмо-приглашение на почту.
 
Пример письма-приглашения 

После перехода на страницу профиля Вы можете поменять пароль в разделе «Изменить пароль (Change Password)». Вы можете придумать новый пароль сами или воспользоваться генерацией пароля, нажав на иконку молнии.
 
Кнопка для генерации пароля 


После внесения изменений, нажмите кнопку «Сохранить (Save)».

Импорт CSV файла

Нажмите ПКМ на "Home" в разделе "Products".

Затем выберите пункт "Product" в разделе "CSV Import".

В появившемся окне выберите файл для загрузки (заполненную форму для поставщиков в формате CSV).

Для импорта файла нужно загрузить необходимые настройки Pimcore. Для того, чтобы загрузить настройки, нажмите на кнопку "Load".

В появившемся окне выберите "Форма пример" и нажмите "ОК".

Для того, чтобы начать процесс импорта, нажмите "Import".

После завершения процесса импорта, Вы получите сообщение. Нажмите кнопку "ОК", чтобы перейти к результатам.

На вкладке "Import Report" находятся результаты последнего процесса импорта. В случае, если при импорте возникли проблемы, в колонке "Message" будет находиться описание ошибки.

Для того, чтобы открыть успешно импортированный товар, нажмите на значке "Open".

Также вы можете проверить наличие импортированных товаров в полном списке, нажав на папку "Home" и проверив их наличие в списке.

Создание нового товара вручную

Создание объкта класса "Товар"

  1. Нажмите ПКМ на "Home" в вкладке "Products"
  2. Выберите пункт "Product" в разделе "Add Object"
  3. Введите название для нового товара (оно будет отображаться в списке слева и должно быть уникальным) и нажмите "ОК"

Заведение основных характеристик товара

Заведение основных значений

  1. Добавить торговую марку
    Для этого открываем созданный товар и на вкладке "Тип товара" заполняем поле "Торговая марка":

    1. нажав на лупу и воспользоваться поиском
    2. Перетащить элемент со значений, папка "Торговая марка"
  2. Заполнить поле "штрих код"
    Согласно требованиям штрих код может присваиваться автоматически либо ввести его вручную.
    1. Для того чтобы штрих код сгенерировался автоматически необходимо соблюдение следующих правил:
      1. На присвоенной к товару Торговой марке должна быть заполнена вкладка "EAN Generation"
        1. поле "Min val"
        2. поле "Max val"
      2. В поле штрих код на вкладке "3- свойства" на товаре поставить цифру "0"
      3. Нажать кнопку "Save". После этого штрих код заполнится автоматически
    2. Заполнить поле "Штрих код" на вкладке "3- свойства"

      !!! Если штрих кода, нет в справочнике "Barcode", то система создаст в нем новую запись
  3. Заполнение поля "Категория ТМ дополнительная"
    Поле категория ТМ дополнительная (CategoryTM_Additional_select) заполняется автоматически после выбора Торговой марки на товаре и нажатии на Save. Значение выбрано с  объекта Торговая марка, поле "trade_mark_category_extra"
  4. Заполнение поля "Семейство товара"
    После того, как было заполнено поле "Категория ТМ дополнительная" и сохранены изменения на товаре формируется выпадающий список значений для поля "Семейство товара". Выбираем значение и нажимаем "Save". Список значений можно добавлять и редактировать в папке "Набор полей и правил", добавляя туда объекты "productSetFieldsRules"
  5. Заполнение поля "Тип товара"
    После заполнения поля "Семейство товара" формируется выпадающий для поля "Тип товара". Выбираем значение и нажимаем "Save". Список значений можно добавлять и редактировать в папке "Набор полей и правил", добавляя туда объекты "productSetFieldsRules"
  6. Заполнение поля "Пол"
    После заполнения поля "Тип товара" формируется выпадающий для поля "Пол". Выбираем значение и нажимаем "Save". Список значений можно добавлять и редактировать в папке "Набор полей и правил", добавляя туда объекты "productSetFieldsRules"
    !!! Значений в поле "Пол" можна выбрать несколько. Для этого используем кнопку "Ctrl"
  7. После заполнения полей в пунктах 1-6 система Pim Core автоматически заполняет следующие поля:
    1. Область применения (для фильтра в им) (productUsageArea)
    2. Область применения (для Тетры) (productUsageAreaTetra)
    3. Категория всех уровней (Category)
    4. Тип товара атрибут (productAttributeType) в работе ( BRD-254 - Автоматическое заполнение поля Тип товара атрибут DONE )
  8.  

    Работа с framework

Согласно требованиям framework на товаре присваивается автоматически согласно сдеющего правила:

  • Если на типе товара в поле Тип фреймворка выбрано "Общий", то значение берем с поля Framework на типе товара.
  • Если на типе товара в поле Тип фреймворка выбрано "Уникальный", то значение берем с поля FrameworkCode в объекте торговая марка.

Перечень всех фреймворков находится в папке "Настройка структуры вариантов", объекты "VariantStructure"

После того как на товар был присвоен framework можна заполнить следующие поля:

  1. Номер тона
  2. Название тона
  3. Код Hex
  4. Объем

Первые 3 поля будут необязательными - внести изменения на Feature and Dev

Формирование выпадающих списков

Для формирвоания выпадающих списков по полям выше необходимо добавлять объекты productVariantStructureMeasure (Связка Framework с кодом структуры) и productVariantStructureBasikMeaning (Непосредственно сами значения), что будут находится в папках:

  • Вариант структура измерения - productVariantStructureMeasure
  • Вариант Структура Базовые Значения - productVariantStructureBasikMeaning

Обязательно значения номер тона, название тона и код HEX должны быть добавлены в соответствующие справочники

Добавление новых значений

Правило: возможно добавление, когда к фреймворку создан объект productVariantStructureMeasure, что определяет код структуры

  1. На вкладке Framework заполнить текстовые поля справа и нажать "Отправить"
  2. Данные поля будут добавлены как новые значения в справочники как неопубликованные:
    1. Номер тона
    2. Название тона
    3. Код HEX
    4. Вариант Структура Базовые Значения
  3. Опубликовать значения в соответствующих справочниках
  4. Завести новые значения на товар через выпадающие списки

Заведение обязательных атрибутов на карточке товара

Для возможности опубликовать товар необходимо заполнить обязательные атрибуты на товаре:

  • артикул
  • Группа ароматов
  • Линия оригинальное название
  • Линия на украинском языке
  • Название товара
  • Название товара укр
  • Название товара рус
  • Дополнительное свойство/действие/эффект, которое должно отобразиться в названии Navision
  • Характер аромата 
  • Ноты
  • СПФ
  • Эффект
  • Покрытие
  • Тип кожи
  • Тип волос
  • Возраст
  • Действие/свойство
  • Ключевые ингредиенты
  • Особенности товара/упаковки
  • Текстура/консистенция 
  • Лимитированный выпуск
  • Розничные рекомендованные цены
  • Срок хранения, мес.
  • № сертификата ГЗ
  • Длина продукта в упаковке, мм
  • Ширина продукта в упаковке, мм
  • Высота продукта в упаковке, мм
  • Вес продукта в упаковке, г
  • Количество в упаковке для продажи
  • Количество в коробке
  • Код ВЭД
  • Состав товара (INCI)
  • Состав товара рус
  • Состав товара укр
  • Способ применения рус
  • Способ применения укр
  • Представленность позиции




Редактирование товара

Для того, чтобы отредактировать товар, откройте его двойным нажатием в разделе "Products".

Для того, чтобы были доступны все поля, нужно, чтобы поле "Тип товара" было заполнено. Если оно пустое, выполните следующие шаги:

Для того, чтобы выбрать тип товара, откройте папку "Тип товара" в вкладке "Values"

Для того, чтобы сделать поиск нужного значения, нажмите на иконку поиска

Затем, в появившемся поле введите название нужного значения и нажмите Enter.

Перетяните выбранный тип товара на поле "Тип товара" в карточке товара

Нажмите кнопку "Save" для сохранения изменений

Нажмите на кнопку "Reload", чтобы перезагрузить карточку товара

После перезагрузки товара в карточке появятся новые вкладки для внесения информации


Перейдем на вкладку "2 - Свойства" для того, чтобы продемонстрировать работу с разными типами данных.

Работа с полями, которые являются ссылками на объекты

В Pimcore все поля, значение которых выбирается из списка, являются объектами и поэтому для их заполнения в карточке товара надо привязать товар к объекту-значению списка. Все такие поля-списки и их значения находятся в правой панели "Значения" ("Values"). Значения для таких полей можно выбрать либо перетягиванием из панели "Значения", либо нажав на иконку поиска (см. Работа с полями, которые являются ссылками на объекты (альтернативный метод)). Покажем заполнение таких полей на примере поля "Торговая марка".

Поле "Торговая марка" (1) - это ссылка на объект, в котором хранится значение. Красный значок навигации (2) указывает на то, что значение можно установить, перетянув объект на поле (Drag-and-drop).

Для того, чтобы найти нужную торговую марку, откройте папку "Торговая марка" (3) в разделе "Values".

Для того, чтобы задать значение, перетяните объект с раздела "Values" на нужное поле.

Работа с полями, которые являются ссылками на объекты (альтернативный метод)

Также вы можете задать значение альтернативным способом. Для этого нажмите на значок поиска (Search).

В появившемся окне Вы увидите список объектов, которые могут быть использованы как значение. Для поиска, введите текст и нажмите кнопку "Search".

Выберите нужное значение с помощью двойного нажатия либо ПКМ и затем "Add".

Проверьте, что значение появилось в разделе "Your Selection" и нажмите "Select".

Работа с текстовыми и числовыми полями

Для того, чтобы задать текстовое или числовое значение, просто введите его в нужное поле.

Работа с выпадающими списками

Для того, чтобы задать значение выбора, откройте список вариантов и выберите нужное значение.

Работа с датами

Для того, чтобы задать значение даты, нажмите на значок календаря и выберите нужную дату.

Работа с изображениями

Для демонстрации работы с изображениями перейдем на вкладку "3 - Контент".

Если изображение уже существует в системе PImcore, откройте раздел "Assets" для просмотра доступных изображений.

Работа с существующими изображениями

Поскольку на полях для изображений также присутствует красный значок навигации, Вы можете перетянуть изображение из раздела "Assets" на нужное поле.

Работа с существующими изображениями (альтернативный метод)

Также Вы можете воспользоваться альтернативным методом выбора. Для этого нажмите на значок меню внутри поля для изображения.

Затем выберите пункт "Search".

Нажмите кнопку "Search" для отображения всех доступных изображений.

Выберите нужное изображение из списка двойным нажатием мыши.

Загрузка новых изображений

Для того, чтобы загрузить файл и сразу же использовать его как значение поля, нажмите на значок меню внутри поля для изображения.

После этого выберите пункт "Upload".

В появившемся окне выберите изображение для загрузки.

После загрузки, изображение появится в нужном поле.

Сохранение изменений

Для сохранения изменений нажмите кнопку "Save".
