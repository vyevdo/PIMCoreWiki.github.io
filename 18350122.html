Краткое описание

Функция экспорта реализована с помощью готового бандла "OutputDataConfigToolkit" и своего созданного банда, который расширяет ее функционал. Экспорт работает с файлами в формате CSV. Для экспорта можно создавать свою конфигурацию (т.е привязку колонок файла к атрибутам объекта) для сохранения настроек при экспорте данных в разные каналы вывода.

Экспорт

Для проведения экспорта нужно проделать следующий набор шагов:

  1. Войти в админ панель: {+}https://brd.nodeart.app/admin+
  2. После входа в панели слева выбрать элемент, с которым мы хотим работать (для экспорта всех товаров - "Home", для товаров определенной категории - эту категорию, для конкретного товара - этот товар).
  3. Затем нажать кнопку "Export to CSV" (Не путать с "CSV Export"):
  4. В окне выбираем тип экспорта - tetra или navision:
  5. Подтверждаем выход из страницы:
  6. Готово:

Настройка экспорта

Для настройки полей для экспорта нужно выполнить следующие шаги:

  1. Выбрать в левой панели папку "Home"
  2. Перейти в раздел "Output Data Configuration"
  3. Выбрать канал вывода, который мы хотим редактировать:
  4. Делаем изменения по тому же принципу, что и в товарной сетке и импорте и нажимаем "Apply":
