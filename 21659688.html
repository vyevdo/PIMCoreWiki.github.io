Brocard : Роль Администратор

Created by Valeriy Kulyk, last modified on Aug 14, 2019


Выполняет все действия предусмотренные интерфейсом пимкор

  • Авторизоваться
  • Изменить пароль
  • Создать учетную запись
    • Поставщика
    • Бренд менеджера
    • Товароведа
    • Администратора
  • Администратор должен работать только со всеми брендами
  • Использовать дефолтную перспективу для данных о продукте
  • Управлять всеми классами у которых будут хранится значения параметров
  • Внести данные о товаре через интерфейс Pimcore
  • Загрузить данные о товаре с помощью .csv
  • Менять статус товара (В работе, подтвержден, опубликован, в архиве) (воркфлоу доступен в pimcore advanced demo)
  • Управлять Custom Layouts
    • Создавать
    • Вносить массовые изменения
    • Применять для типов товаров
  • Иметь доступ к дропдаун меню(Actions) "доступные операции по товару"
    • Если товар в работе (не подтвержден Бренд Менеджером или Товароведом)
      • Сохранить товар
      • Редактировать товар
      • Подтвердить товар
    • Если товар подтвержден Бренд Менеджером или Товароведом
      • Отменить подтверждение товара
      • Опубликовать товар
      • Принять запрос на изменение товара
        • менять штрихкод на данном этапе уже нельзя 
    • Если товар опубликован Бренд Менеджером или Товароведом
      • Экспортировать товары в .csv
      • Отменить публикацию путем автоматического переноса его в статус "в работе"
      • Принять запрос на изменение товара
        • Менять штрихкод на данном этапе уже нельзя 
      • Заархивировать товар
    • Если товар заархивирован Товароведом
      • Разархивировать товар
