Brocard : Требования товар

Created by Valeriy Kulyk, last modified on Aug 14, 2019

Товар должен иметь следующие особенности:

  • Открыть товар
  • Сменить статус товара
  • Изменить поле
  • Bulk Change 
  • Должен иметь возможность указать определенное кол-во атрибутов
  • Атрибуты определяются классом товара
  • По умолчанию для администратора для всех товаров доступны все возможные атрибуты
  • Доступность атрибутов
    • Обзятальность заполнения атрибутов зависит от типа выбранного товара
  • Роли пользователя
    • Механизм реализации - Custom Layouts
  • Некоторые поля должны иметь возможность валидации вводимых значений 
    • Ждем уточнений
  • Для каждого атрибута определяется тип, доступность, значение по умолчанию, необходимость валидации вводимых значений с указанием метода валидации. 
  • Все поля в Custom Layouts должны быть сгрупированы в 4 вкладки
    • Вкладка 1  содержит тип товара
    • Вкладка 2  содержит все атрибуты товара доступные для выбраного типа товара и данной роли кроме параметров контента, которые содержатся в третей вкладке
    • Вкладка 3 содержит атрибуты контента ( все фото, видео, текстовое описание для веб сайта, инструкции по применению) (все фото и видео хранятся в ассетсах с дальнейшей перелинковкой с карточкой товара)
    • Вкладка 4 содержит контроль данных - содержит автоматические рассчитанные параметры по контролю качества данных товаров
      • Процентность заполнения и тд. Ждем полный список. 
  • Товары которые между собой отличаются одним атрибутом 
  • Выполнить операцию - создать варианты (обьединение товаров которые отличаются штрихкодом или одним или несколькими параметрами групируются в основной товар и его варианты. (Пример помада лореаль где различие будет только в параметре - цвет) (  опционально и зависит от стоимости. )
  • Содержать текстовое поле - поставщик
  • По умолчанию иметь только меню actions

    • Просмотр списка товаров
      • Запретить inline edit. (редактировать можн только в карточке товара)
    • Просмотр карточки товаров
      • По умолчанию иметь только меню actions
