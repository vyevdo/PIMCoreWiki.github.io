Brocard : Импорт кодов Тетра из FTP

Created by Dmytro Svietnoi, last modified on Sep 21, 2021


На сервере внутри лежат csv файлы, формат которых

GoodID,BarCode,Articul
60002,3298837373511,
60008,037361000042,41252
60010,037361001520,9452

При загрузке CSV происходит попытка найти товар по паре значений BarCode, Articul и присвоить им код Тетры.
Если артикула нет либо он не обнаружен - ищем просто по баркоду.
Кода Тетры которые уже есть в системе повторно не обрабатываются.

Настройки ftp сервера находятся в пиме /Settings/Website Settings:

ftp_hosttest.site.ua
ftp_useruser
ftp_passwordpassword
ftp_directory/TETRA
notifier_group_id123

Файлы должны находится в директории IN. Обработанные файлы автоматически перемещаются в директорию PROCESSED

Отчёты об ошибках и сгенерированные в процессе файлы рассылаются всем юзерам, которые находятся внутри папки, id которой указывается параметром notifier_group_id.
