Краткое описание

В Pimcore взаимодействие со списком товаром выполняется при помощи сетки объектов (Object Grid). Сетки можно создавать, редактировать отображаемые колонки и привязывать сетку как стандартную к объекту. В колонках сетки можно настроить фильтрацию либо выполнить массовое редактирование.

Отображение сетки

Для того, чтобы выполнить вышеуказанные действия, сначала всегда нужно проделать следующий набор шагов:

  1. Войти в админ панель: {+}https://brd.nodeart.app/admin+
  2. После входа в панели слева выбрать объект (категорию), с которой мы хотим работать:
     
  3. В появившейся вкладке выбрать раздел "Children Grid" (Сетка дочерних элементов): 
  4. Если это промежуточная категория (т.е внутри неё не только товары, но и другие категории), то выбираем тип объекта (Категория или Товар), который нас интересует:
  5. Убираем галочку "just direct children" (Только непосредственные дочерние элементы) - это позволяет нам увидеть товары вложенных категорий:
  6. Видим сетку с вложенными товарами, при необходимости выбираем нужную сетку.

Настройка колонок сетки

  1. Открыть отображение сетки
  2. Нажать на кнопку "Grid options":
     
  3. Видим следующее окно, в нем настраиваем сетку и сохраняем:
             
                    

1 - Поле для поиска атрибутов
2 - Атрибуты объекта (товара), которые можно добавить в сетку, если переместить их мышкой в правую часть окна
3 - Атрибуты (колонки), которые будут отображаться в сетке. Их можно удалить нажав ПКМ и выбрав "Delete"
4 - "Apply" - кнопка для сохранения настроек

Массовое редактирование

  1. Открыть отображение сетки
  2. Выбрать нужную колонку, и один из пунктов:
    1. "Batch edit all" - изменить данные всех товаров в сетке
    2. "Batch edit selected" - изменить данные выделенных товаров
    3. "Batch append all" - добавить значение для мультивыбора всех товаров в сетке
    4. "Batch append selected" - добавить значение для мультивыбора выделенных товаров

  1. Вводим информацию в появившемся окне
  2. Готово.
