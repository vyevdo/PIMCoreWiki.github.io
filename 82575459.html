Цели

  • Передать данные о товаре для стикеровки в систему Labels

AS IS

  1. Тетра Office → MS Access
  2. MS Access→ Excel.xlsm
  3. Excel.xlsm→ Стикер

Список полей, которые необходимы для стикерования:

  • Автозаполнены из Тетры Office:
    • ТМ
    • Линия
    • Название
    • Артикул
    • Объем
    • Штрих-код
    • Название для стикера
    • Производитель
    • Страна производства
    • Состав
  • Заполняются вручную (информация предоставляется поставщиком)
    • Срок годности (срок хранения)
    • Примечание

Предположения

  • Штрих код считаем уникальным полем
