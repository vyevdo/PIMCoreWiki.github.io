Цели

  • Настроить работу с вариантами в PIM Core

As it

  1. Объединить кодом Navision товары, что нужно добавить в единую карточку товара в Excel файл
  2. Загрузить данный файл в Navision
  3. В случае ошибок с объединением процесс их исправления достаточно трудозатратный

Предположения

Для объединения  в варианты

  • Должны совпадать:
    • ТМ
    • Линия
    • Тип товара
    • Эффект
    • Объем различная
  • Для парфюмерии и ухода (Fragrance, Niche, care) - разный объем
  • Для MakeUp - одинаковый объем, разные оттенки
  • могут быть исключения в виду разнообразия продуктов
