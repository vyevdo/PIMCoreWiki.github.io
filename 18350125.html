Краткое описание

Настройка доступа реализовывается с помощью использования ролей. С помощью ролей мы можем разграничить пользователей с разными правами. Написанный код для решения задачи использует две роли: поставщик (supplier) и бренд-менеджер (brand_manager).

Создание нового пользователя

Для того, чтобы создать нового пользователя, нужно проделать следующий набор шагов:

  1. Войти в админ панель: {+}https://brd.nodeart.app/admin+
  2. После входа в панели слева выбрать раздел "Users":
  1. Затем нажать ПКМ на "All users" и выбрать "Add user": 
  2. Вводим имя пользователя:
     
  3. Затем в окне справа вносим всю нужную информацию и нажимаем "Save": 
  4. Готово.

Поставщики

В поставщиков есть право взаимодействовать только с теми товарами, которые они создали или импортировали. Для создания поставщика, после создания пользователя надо выбрать роль "supplier", и в пользователя автоматически будет выполняться проверка на вышеуказанные ограничения.

Бренд-менеджеры

В бренд-менеджеров есть право взаимодействовать только с товарами тех брендов, с которыми они работают. Для создания бренд-менеджера, пользователя нужно назвать таким образом:
никнеймбренд1бренд2_...
Например:

  1. И. Иванов, который работает с L'oreal, будет называться ivanov_loreal
  2. П. Петров, который работает с Givenchy и United Colors of Benetton (в которого владелец - Antonio Puig S.A.), может называться petya13_givenchy_antoniopuigsa


Также этим пользователям нужно присвоить роль "brand_manager".
