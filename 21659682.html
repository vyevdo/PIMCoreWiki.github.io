Brocard : Роль Поставщик

Created by Valeriy Kulyk, last modified on Aug 12, 2019

Как поставщик я должен иметь возможность:

  • Авторизоваться
  • Изменить пароль
  • Работать только со своими брендами(присваивается админом при регистрации поставщика, может быть расширен в будущем)
  • использовать кастомную перспективу для данных о продукте с корнем Products из Data Objects, при этом медиа файлы по продукту должны сохраняться в Products из Assets. Таким образом фото не должны располагаться в карточке товара, а должна быть проброшена связь из карточки товара к файлам хранимым в ассетах.
  • Внести данные о товаре через интерфейс Pimcore
  • Загрузить данные о товаре с помощью .csv
  • Загрузить данные о товаре с помощью API 
  • Видеть статус товара (В работе, подтвержден, опубликован, в архиве) (воркфлоу доступен в pimcore advanced demo)
  • Иметь доступ к дропдаун меню(Actions) "доступные операции по товару"
    • Если товар в работе (не подтвержден Бренд Менеджером или Товароведом)
      • Сохранить товар
      • Редактировать товар
    • Если товар подтвержден Бренд Менеджером или Товароведом
      • Запрос на изменение товара
        • пользователю необходимо будет внести изменения, а затем нажать на кнопку предложить изменения (запросить изменение штрихкода на данном этапе уже нельзя ).
    • Если товар опубликован Бренд Менеджером или Товароведом
      • Запрос на изменение товара
        • пользователю необходимо будет внести изменения, а затем нажать на кнопку предложить изменения (запросить изменение штрихкода на данном этапе уже нельзя ).
  • Получать уведомления на почту о изменениях товара, включая массовые изменения. 
  • Получать уведомления в телеграмм (опционально)
