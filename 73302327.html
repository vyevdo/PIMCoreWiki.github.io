Brocard : Инструкция по работе с импортом (depricate)

Created by Dmytro Svietnoi, last modified on Nov 22, 2022

Краткое описание

В Pimcore есть встроенная функция импорта, которая работает с файлами в формате CSV. Для импорта можно создавать свою конфигурацию (т.е привязку колонок файла к атрибутам объекта) для сохранения настроек при будущих импортах файлов с одинаковой структурой

Импорт

Для проведения импорта нужно проделать следующий набор шагов:

  1. Войти в админ панель: https://pim.brocard.ua/admin/
  2. После входа:
    1 шаг:  нажать на иконке "Datahub"
    2 шаг:  выбрать тип объекта (категорию или продукт), который мы хотим импортировать
    3 шаг:  перейти на вкладку "Data Source"
    4 шаг:  выбрать файл для импорта csv с необходимой кодировкой и разделителями


  3. Затем перейти на вкладку "Execution" и нажать кнопку "Start"
    5 шаг: переход на вкладку "Execution"
    6 шаг: нажимаем "Start"
     

  4. После чего, по истечение небольшого промежутка времени Ваш файл будет автоматически обработан и логи этого события будут находиться в следующей вкладке этого же окна "Import logs":
