Цель

  • Проговорить, занимается ли маркетинг медиаконтентом
  • Какие атрибуты используются сейчас, нужны ли какие-то дополнительные атрибуты

Результаты встречи

  • Медиаконтентом (фото, видео) для сайта, каталогов, журнала подразделение маркетинга Брокарда не занимается
  • Отдел дает запрос на заведение таких товаров, как рекламные, подарочные, упаковка (пакеты) в отдел товароведения, следуя принятым инструкциям товароведов
  • Отдел пользуется системой Navision
  • Никакой новой информации по товару подразделение не добавляет, не заводит
  • На текущий момент Ярослава самостоятельно делает некоторые отметки в Navision в карточке товара, в будущем эти правки должны вносить товароведы (см.файл во вложении). Эти настройки предназначены для корректной работы акций в магазинах и на сайте
  • Помимо требований в файле, еще одно: для карточки товара, которая относится к акционным картам (пластик со ШК) не создавать код варианта, это удобнее и проще
  • В маркетинге формируется объявление для акций на сайте (баннеры рекламные)

Вопросы

  • Вносит ли Александр Бондаренко (склад) и Светлана Будзан дополнительно какие-то настройки, атрибуты по товару?
  • Где отмечаются дополнительные настройки при отгрузке товара (при вкладывании пробников, журналов), возможно Navision? 
  • Связан ли кто-то из администрирования сайта с медиаконтентом?

Обязательные настройки карточки товара для успешной работы маркетинговых процессов:

  1. Для всех товаров, которые не предназначены для продажи (товары или карты, которые выдаются на ПОС (рабочее место кассира) в качестве дополнительной выгоды для клиента)
    1. Настройки карточки товара в разделе POS:
      1. Поле ВведитеЦену - выбрать Не обязательно
      2. Поле Введите кол-во - выбрать Не обязательно
      3. Предлож. кол-во на POS - 0.00
      4. Разрешена нулевая цена - поставить галочку
    2. В таблице Item Status (10001403) настроить статус товара Block Marketing Sales:
      1. Code - BM_SALE, Description - Block Marketing Sales: поставить галочки в Block Discount, Block Periodic Discount, Заблокировать GWP
  2. Для товаров, которые относятся к продаваемым или акционным картам, необходимо настроить трассировку товара.
    1. Если карта имеет уникальный штрихкод для каждой карты (пластик) необходимо указать код трассировки товара GCSERIAL (трассировка по серийному номеру)
    2. Если карта имеет единый штрихкод для каждой карты (пластик) необходимо указать код трассировки товара GCLOTNEW (трассировка по номеру партии)
    3. Для любого вида карты необходимо установить признак «Отключение трекинга для ИМ = ДА»
  3. Для товара, который предназначен для автоматического добавления в заказ в качестве дополнительной выгоды при выполнении условия покупки (корзина ИМ Brocard.ua), необходимо указать Код назначения товара = PROMOTION и код типа товара = GIFTS
  4. Для карточки товара, которая относится к акционным картам (пластик со ШК) не создавать код варианта
