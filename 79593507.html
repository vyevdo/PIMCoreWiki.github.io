Бизнес правила

При загрузке товаров в Navision им присваивается код Navision("Vendor Item No."), который может объединять несколько кодов Тетры ("GoodID Tetra"). Объединение происходит по вариантам товаров.

Уникальным значением в Navision будет Код Navision + Код Варианта


Логика работы с вариантами:

В Navision есть два вида карточек:

  • Иерархия Skin, Parfume
  • Make up

В Иерархию Skin нельзя загрузить два одинаковых объема(например по 30 мл), Есть набор атрибутов, который показывает уникальность товара. Штрих коды все разные

В Иерархию Make up нельзя загрузить два одинаковых тона

Логика загрузки наборов в Navision происходит в ручную
