Цели

  • Передать данные о товаре для стикеровки в систему Labels 2.0

AS IS

  1. Один поставщик (он пока единственный) высылает данные в формате DBF (MS Access) и один Excel файл по шаблону
  2. Данные хранятся в базе MS Access
  3. MS Access→ MySQL
  4. MySQL→ Excel.xlsm
  5. Excel.xlsm→ Стикер

Список полей, которые необходимы для стикерования (все поля предоставляются поставщиком):

  • Артикул - первые 4 символа артикула, который указан на товаре
  • Марка товара
  • Название оригинальное
  • Название на украинском языке
  • Объём
  • Состав (ингредиенты)
  • Страна
  • Дата производства
  • Срок хранения
  • Меры предосторожности
  • Адрес производителя
